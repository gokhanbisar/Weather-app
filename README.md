# 🌤 Weather App

Bu proje, Python ve **CustomTkinter** kullanılarak oluşturulmuş basit bir hava durumu uygulamasıdır. Kullanıcı bir şehir girerek OpenWeatherMap API üzerinden güncel hava durumunu görebilir.

---

## ✨ Özellikler

- 🏙 Şehir adı girerek güncel hava durumu bilgisi alabilme
- 🌈 Hava durumuna göre görsel ikonların otomatik değişmesi
  - ☀️ Clear  
  - ☁️ Clouds  
  - ❄️ Snow  
  - 🌧 Rain
- 🌡 Minimum ve maksimum sıcaklık bilgisi gösterimi
- 🖼 Basit ve kullanıcı dostu arayüz (CustomTkinter + Canvas)
